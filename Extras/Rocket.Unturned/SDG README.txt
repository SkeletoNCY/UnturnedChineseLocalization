RocketMod is a popular server-side plugin framework which expands the administrative toolset and wraps the Unturned APIs in a user-friendly manner.

On the 20th of December 2019 Sven Mawby "fr34kyn01535" and Enes Sadık Özbek "Trojaner" officially ceased maintenance of RocketMod.
They have kindly released the source code under the MIT license. Read their full farewell statement here:
https://github.com/RocketMod/Rocket/blob/master/Farewell.md

This build of RocketMod bundled with the core game exists to fix issues that cropped up in the final release and preserve compatibility.
Read full documentation here: https://github.com/SmartlyDressedGames/U3-Docs/blob/master/RocketMod.md
